What is shadcn/ui? ▼

shadcn/ui is a collection of re-usable components built using Radix UI and Tailwind CSS. It's not a component library, but rather a set of components you can copy and paste into your apps.

How is it different? ▼

Unlike traditional component libraries, shadcn/ui gives you full ownership of the code. You can customize every aspect to fit your needs without fighting with library constraints.

Key Features ▼
  • ✓ Fully customizable components
  • ✓ Built with accessibility in mind
  • ✓ Dark mode support
  • ✓ TypeScript support
